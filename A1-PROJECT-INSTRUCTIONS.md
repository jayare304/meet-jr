## The Project
**Meet JR** is an interactive portfolio at `jrcrosby.info` that lets visitors experience JR's professional journey through conversation with Claude, rather than just reading a static resume.

**Two paths:**
1. **Download Resume** - PDF for traditional workflows
2. **Meet JR** - Claude-powered chat through JR's story

**Tech:** HTML/CSS/JS frontend, Node.js + Anthropic API backend, Vercel hosting

**GitHub:** github.com/jayare304/meet-jr
